# Country-Information
This is a console app in which you can type a country and you will get information about it. This program uses the
library "countryinfo".

## Information given:
- Language/s;
- Capital city;
- Population;
- Currency/ies
- Native's name
- Resident's name;
- Area(kmÂ²);
- Province
- Region;
- Borders;
- Calling code;
- Time zone;
- Domain;
- More information(link to wikipedia).
